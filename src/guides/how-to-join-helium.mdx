# How to Join Helium

Helium is a decentralized wireless network that rewards users for sharing connectivity.

## Steps to Get Started

1. Buy a Helium-compatible hotspot
2. Register it via the Helium App
3. Start earning HNT tokens

> Tip: Choose an area with high foot traffic or IoT demand!
